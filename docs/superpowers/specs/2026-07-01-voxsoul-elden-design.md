# VoxSoul — 艾尔登法环玩法 Luanti 子游戏设计规格

**日期**：2026-07-01  
**状态**：Approved（2026-07-01 自主确认）  
**形态**：Luanti 子游戏（Game Pack），标准客户端运行，不改引擎  
**目标 Luanti 版本**：5.13+

---

## 1. 项目概述

VoxSoul 是基于 [Luanti](https://github.com/luanti-org/luanti) 开发的魂类动作 RPG 子游戏，玩法高度致敬《艾尔登法环》。v1 为 **纯单人** 小型 Demo，预计 **1–2 小时** 通关体验。

### 1.1 优先级

| 优先级 | 系统 |
|--------|------|
| P0 | 战斗手感（耐力、闪避、格挡/弹反、Poise） |
| P1 | Boss 战（多招式、阶段切换） |
| P2 | 开放探索（小区域连通、赐福点、隐藏要素） |
| P3 | 角色成长（属性升级、武器、护符） |
| — | 叙事/ NPC 对话（v1 不做） |

### 1.2 关键决策

| 维度 | 选择 |
|------|------|
| 实现路线 | 方案 A：自研魂类战斗框架 |
| 视角 | 第三人称越肩，v1 不提供第一人称切换 |
| 视觉 | 混合：体素世界 + mesh 玩家/Boss |
| 世界观 | 高度致敬法环（粉丝向，非官方，不商用） |
| 地图 | 手工搭建，不使用随机 mapgen |

### 1.3 版权与发布限制

本项目为 **非官方粉丝项目**。v1 使用法环致敬命名与造型，仅适合个人或小范围试玩。若后续公开发布至 ContentDB 或商用，**必须**替换为原创 IP 或取得授权。仓库内须包含 `FAN_PROJECT.md` 声明。

### 2.2 模块依赖

```
voxsoul_core
  ├── voxsoul_camera
  ├── voxsoul_entity
  │     └── voxsoul_combat
  │           ├── voxsoul_player
  │           │     ├── voxsoul_grace
  │           │     └── voxsoul_items
  │           ├── voxsoul_boss
  │           └── voxsoul_ui
  └── voxsoul_world（依赖 boss、grace）
```

### 2.3 技术基线

- **Luanti 5.13+**：使用 `player:set_camera({mode = 'third'})` 锁定第三人称；`set_eye_offset` 实现越肩偏移
- **不依赖** Advanced Fight Library（Beta，API 不稳定）
- **Hitbox**：自研扇形/圆形/矩形近战判定
- **mesh 实体**：玩家与 Boss 使用 animated mesh；加载失败时回退为半透明占位方块

## 3. 战斗系统（`voxsoul_combat`）

### 3.1 输入映射

| 按键 | 动作 |
|------|------|
| 鼠标左键 | 轻攻击 |
| Shift + 左键 | 重攻击 |
| 鼠标右键（按住） | 格挡 |
| 格挡中受击前 0.25s 内松开右键 | 弹反 |
| Space | 跳跃 |
| Space + 移动方向 | 闪避翻滚 |
| E | 交互（赐福、拾取卢恩堆） |
| Z (zoom) | 锁定/取消锁定最近敌人 |

`voxsoul_core` 禁用默认挖掘、放置、投掷等操作。

### 3.2 耐力（Stamina）

- 初始最大耐力：100（可通过耐力属性提升）
- 恢复延迟：攻击/翻滚结束后 0.8–1.0s 开始恢复
- 恢复速度：约 35 点/秒
- 耗尽惩罚：无法攻击/闪避/格挡，移速 -30%，恢复至 20% 以上解除

| 动作 | 耐力消耗 |
|------|----------|
| 轻攻击 | 12 |
| 重攻击 | 28 |
| 闪避 | 22（连续 3 次内每次 +5） |
| 格挡 | 8/秒 + 受击额外消耗 |
| 弹反失败 | 15 |
| 弹反成功 | 0（并恢复 30 耐力） |

### 3.3 生命与 Poise

- 初始 HP：400（活力属性每级 +20）
- 受击硬直：轻击 0.2s，重击 0.5s
- Poise 归零：1.2s 大硬直，处决加成 +50% 伤害
- 死亡：在最近赐福点复活，失去 50% 持有卢恩

| 攻击类型 | Poise 伤害 |
|----------|------------|
| 轻攻击 | 8 |
| 重攻击 | 25 |
| Boss 招式 | 15–40（按招式定义） |

Poise 非硬直状态下约 5%/秒 恢复。

### 3.4 攻击判定

- 前摇结束后，在玩家前方扇形区域检测（半径 2.5m，角度 90°）
- 伤害 = 基础伤害 × 武器倍率 - 目标防御
- 轻攻击：基础伤害 35，前摇 0.3s
- 重攻击：基础伤害 70，前摇 0.7s

### 3.5 闪避

- 条件：Space + 移动输入 + 耐力 ≥ 22 + 非硬直/非攻击中
- 动画 0.6s，无敌帧 0.1s–0.5s（共 0.4s）
- 四方向翻滚（相对相机/角色）

### 3.6 格挡与弹反

**格挡**：移速 -50%；受击消耗耐力 = 伤害 × 0.6；耐力足够则伤害 -80%；不足则破防（1.0s 大硬直 + 全额伤害）。

**弹反**：格挡状态下，敌人攻击命中前 0.25s 内松开右键 → 敌人 2.0s 大硬直。

### 3.7 锁定目标

- Q 锁定 15m 内最近敌对实体 → **实现为 Z (zoom)**
- 锁定时相机朝向目标（yaw/pitch）；**WASD 相对目标 strafe 环绕**（Phase E2）
- 目标死亡或超出 20m 自动取消

### 3.8 玩家战斗状态机

状态：`Idle` → `Attacking` / `Blocking` / `Dodging` / `Locked` / `Hitstun` / `GuardBreak` / `Parry`

优先级（高 → 低）：`Hitstun` > `GuardBreak` > `Dodging` > `Attacking` > `Blocking` > `Idle`

### 3.9 v1 不实现

- 魔法/祷告
- 双持/切换武器（固定起始武器 + 1 把掉落武器）
- 背刺/跳跃攻击
- 复杂属性软上限公式

## 4. Boss 系统（`voxsoul_boss`）

### 4.1 框架

Boss 由数据驱动招式表 + AI 状态机定义：

```lua
{
  id = "tree_sentinel",
  name = "大树守卫",
  max_hp = 800,
  max_poise = 120,
  phases = {
    { threshold = 1.0, attack_pool = {"halberd_sweep", "shield_bash", "horse_stomp"} },
  },
  attacks = { /* 前摇、活跃帧、伤害、hitbox、telegraph */ },
}
```

**AI 状态**：`Idle` → `Chase` → `Attack` → `Recovery`；HP 低于阈值 → `PhaseTransition` → `Enraged`

- 仇恨：进入区域触发；赐福复活后不自动仇恨
- 招式选择：加权随机，连续同招权重递减
- 脱战：玩家离开区域 5 秒后 Boss 回满 HP（保留阶段）

### 4.2 Boss 1：大树守卫（可选）

- **位置**：开阔道中央
- **HP** 800，**Poise** 120，单阶段

| 招式 | 前摇 | 伤害 | 要点 |
|------|------|------|------|
| 长戟横扫 | 1.2s | 45 | 扇形，侧向闪避 |
| 盾牌猛击 | 0.8s | 35 | 可弹反 |
| 战马抬蹄 | 1.5s | 50 | 地面 AOE，后撤闪避 |

- **奖励**：800 卢恩 + 护符「黄金树庇佑」（+5 最大 HP）

### 4.3 Boss 2：玛尔基特（必打）

- **位置**：风暴关卡
- **HP** 1200；阶段 2 于 50% HP 触发

**阶段 1**

| 招式 | 前摇 | 伤害 |
|------|------|------|
| 手杖横扫 | 0.6s | 30（二连） |
| 尾击 | 1.0s | 40 |
| 跳劈 | 1.4s | 55 |
| 光刃投掷 | 0.5s | 25 |

**阶段 2**：攻速 +30%；追加连续光刃（3 连）；新招「天降重击」（2.0s 前摇，大范围 AOE）

- **奖励**：3000 卢恩 + 武器「弯刀」

### 4.4 Boss 3：接肢鬼婆（地牢 Boss）

- **位置**：接肢墓最深处
- **HP** 600，**Poise** 50，单阶段

| 招式 | 前摇 | 伤害 |
|------|------|------|
| 抓击 | 0.5s | 25 |
| 翻滚碾压 | 1.0s | 40 |
| 范围喷吐 | 1.8s | 35 |

- **奖励**：1500 卢恩；击败即 Demo 通关

## 5. Demo 区域：「宁姆格福残页」

手工搭建连通小地图，预计 1–2 小时通关。

### 5.1 区域流程

```
引导废墟（教程）
  → 门旁赐福
  → 开阔道（大树守卫 + 失乡骑士 ×2）
  → 风暴关卡前赐福
  → 玛尔基特（关卡 Boss）
  → 关卡后赐福
  → 接肢墓（接肢怪 ×3 + 接肢鬼婆）
```

### 5.2 赐福点（4 个）

| 赐福 | 解锁条件 |
|------|----------|
| 门旁赐福 | 默认 |
| 风暴关卡前赐福 | 靠近激活 |
| 关卡后赐福 | 击败玛尔基特 |
| 接肢墓赐福 | 进入地牢 |

**功能**：休息（回满 HP/耐力，重置普通敌人）、升级、赐福间传送

### 5.3 探索要素

- 1 条隐藏捷径（地牢内通玛尔基特后方）
- 2 处拾取道具（卢恩碎片、失力战灰 ×3）
- 无 NPC 对话

### 5.4 普通敌人

| 敌人 | 数量 | HP | 卢恩 |
|------|------|-----|------|
| 失乡骑士（弱化） | 2 | 150 | 100 |
| 接肢怪 | 3 | 80 | 50 |

## 6. 成长系统（`voxsoul_player` + `voxsoul_items`）

### 6.1 卢恩

- 死亡失去持有卢恩的 50%（向下取整），在死亡位置生成可拾取「卢恩堆」（600s 超时消失）
- 赐福休息不丢失卢恩
- 上限 999999（超出截断）

### 6.2 属性升级

在赐福点消耗卢恩升级；每项升级花费 = `当前等级 × 100` 卢恩；单属性上限 20 级。

| 属性 | 效果 |
|------|------|
| 活力 | 最大 HP +20/级 |
| 耐力 | 最大耐力 +5/级 |
| 力量 | 物理攻击 +3%/级 |
| 敏捷 | 闪避耐力 -2%/级；攻击 +1%/级 |

### 6.3 装备（v1 简化）

| 槽位 | 内容 |
|------|------|
| 武器 | 起始「直剑」；「弯刀」由玛尔基特掉落 |
| 护符 ×1 | 「黄金树庇佑」（大树守卫掉落，+5 HP） |

| 武器 | 伤害倍率 | 攻速 |
|------|----------|------|
| 直剑 | 1.0× | 标准 |
| 弯刀 | 0.85× | 快 20% |

无独立护甲/盾牌槽；格挡为通用右键动作。

## 7. UI / HUD（`voxsoul_ui`）

### 7.1 常驻 HUD

- 左下：HP 条（红）、耐力条（绿，耗尽闪烁）
- 右下：卢恩计数
- 锁定目标时：目标脚下红色标记圈

### 7.2 Boss 血条

- 屏幕顶部：Boss 名 + 血条
- 阶段切换：血条闪烁 + 名称后缀（如「— 恶兆形态」）

### 7.3 赐福菜单

休息 / 升级 / 传送 / 离开

### 7.4 死亡画面

「YOU DIED」+ 失去卢恩数 + 复活赐福点名；2 秒后传送

| 已击败 Boss | world meta `voxsoul:bosses_defeated` |

## 9. 错误处理

| 场景 | 处理 |
|------|------|
| 攻击动画中登出 | 下次登录重置 Idle，HP/耐力保持 |
| Boss 战逃出区域 | 脱战，5 秒后 Boss 回满 HP |
| 玩家卡进体素 | `/voxsoul unstuck` 传送最近赐福 |
| 动画状态冲突 | 按状态优先级强制仲裁 |
| mesh 加载失败 | 半透明占位方块 + 日志警告 |
| 卢恩溢出 | 截断至 999999 |
